Features
- **Daten-Objekte:** Aufgaben werden als strukturierte JS-Objekte verwaltet.
- **Priorisierung:** 3-stufiges System mit Farbkodierung.
- **Interaktive Priorität:** Ändern der Prio durch Klick auf den farbigen Rand (Rotation).
- **Smart Sorting:** Sortierung per Button nach Wichtigkeit.
- **Status-Filter:** Filtern nach "Alle", "Offen" oder "Erledigt".
- **Fortschrittsanzeige:** Dynamischer Balken basierend auf erledigten Aufgaben.
- **LocalStorage:** Deine Daten bleiben auch nach dem Schließen des Browsers erhalten.
